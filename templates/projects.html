Austin Recycling

Austin Recycling helps users to find the nearest facilities that accept the materials that the users wish to recycle in Austin Area. Thanks to City of Austin, the app is using RESTful APIs to get the list of facilities that can process those items.

This app started as a class project together with two other classmates. The website for the project is available here.

A few enhancements have been made to improve the performance and usability of the app, it includes using Retrofit HTTP library, added landscape layout and dual-pane view.
